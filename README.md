# :sunny: Wevid  - Synchronized YouTube Player built in react

> https://wevid.online/

> Synchronized YouTube player that let friends watch videos together regardless of geograpic location

![Example screenshot](screeny.PNG)

### Technologies
* React.js
* CSS
* Node.js
* Websockets
* Express
